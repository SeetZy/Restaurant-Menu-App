# Restaurant-Menu-App

Programma, kas ļauj tev ievadīt ēdiena vārdu, aprakstu un cenu, un kas saglabā šo informāciju datubāzē!
Lapā arī tiek parādīti pēdējie ievadītie ēdieni un saraksts ar ievadītajiem ēdieniem.
